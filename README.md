## 如何运行我们的 Demo
本仓库包含了 Flutter(支持安卓、iOS) 版本的腾讯云媒资托管Flutter SDK 的 Demo，您可以下载Demo来进行体验。
* 使用android Studio 打开项目根目录运行安卓或ios，
* 使用android Studio 打开./android 运行安卓版本demo。
* 使用xcode 打开./ios 运行ios版本demo。
